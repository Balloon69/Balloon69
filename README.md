# MyPortfolio

เว็บพอร์ตโฟลิโอส่วนตัว สร้างด้วย HTML, CSS และ JavaScript

## การติดตั้ง

วิธีติดตั้งโปรเจกต์นี้บนเครื่องของคุณ

### ความต้องการของระบบ

Node.js เวอร์ชัน 18 ขึ้นไป และ npm หรือ yarn

### ขั้นตอนการติดตั้ง

Clone repository นี้ จากนั้นรัน `npm install` และ `npm start`

## ฟีเจอร์หลัก

รายการฟีเจอร์ที่มีในโปรเจกต์

### หน้าแรก (Home)

แสดงข้อมูลแนะนำตัวและ profile

### ผลงาน (Projects)

รวบรวมโปรเจกต์ที่เคยทำทั้งหมด

### ติดต่อ (Contact)

ฟอร์มสำหรับติดต่อกลับ
